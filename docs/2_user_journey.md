# ユーザージャーニー

以下では、主要なユーザー行動のフローを示します。

```mermaid
flowchart LR
  A[ルーム一覧画面表示] -->|空きルーム選択| B[ルーム参加]
  B --> C{ホスト権限あり?}
  C -->|はい| D[ホスト待機画面]
  C -->|いいえ| E[参加者待機画面]
  D & E --> F[全員参加後「ゲーム開始」ボタン押下]
  F --> G[TetrisGame コンポーネント表示]
  subgraph ゲームプレイ
    G --> H[ブロック操作 (←→↓回転)]
    H --> I[ライン消去判定]
    I --> J[ペナルティ生成→相手盤面に送信]
    J --> H
  end
  G --> K[ゲーム終了：リザルト画面表示]
  K --> A[ルーム一覧画面に戻る]
